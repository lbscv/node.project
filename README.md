# Node Project Lars Biczysko

## Beschrijving
Dit project is een **database-driven REST API** gebouwd met **Node.js**, **Express** en **MongoDB**.  
De API bevat twee entiteiten (**Categories** en **Products**) met volledige CRUD-functionaliteit, basisvalidatie, zoekfunctionaliteit en pagination.

Op de root van de applicatie (`/`) bevindt zich een HTML-pagina met documentatie van alle beschikbare endpoints.

## Installatie & opstarten

1. Clone de repository:
    (in bash)
    git clone <GITHUB_REPO_URL>
2. Ga naar project naam
    (in bash)
    cd node.project
3. Download dependencies
    (in bash)
    npm install
    tailwindcss extensie downloaden
4. Start MongoDB lokaal op
    mongodb://127.0.0.1:27017/supermarkt
5. Start server
    (in bash)
    node index.js
6. open in browser via de link dat tevoorschijnt komt
